    <h1 class="h2">Reliable metadata for research software</h1>
    <p class="lead mb-3">
      ELIXIR Research Software Ecosystem (RSEc) curates trusted, cross-linked metadata for computational biology tools so you can discover, validate, and integrate software with confidence.
    </p>
    <ul class="mb-3">
      <li>Single entry point for metadata from <a href="/sources">major registries</a>.</li>
      <li>Open, version-controlled, and <a href="/licence">reusable</a> by design.</li>
      <li>Purpose-built for researchers, curators, and tool developers.</li>
    </ul>
